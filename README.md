# 🔥 Demon Slayer CEP Search

## 📖 Sobre o Projeto

Este projeto consiste em uma aplicação web para consulta de CEPs desenvolvida com **HTML, CSS e JavaScript**, utilizando uma API externa para localizar e exibir informações de endereço a partir de um CEP informado pelo usuário.

Como diferencial, a interface foi personalizada com uma temática inspirada no universo de **Demon Slayer (Kimetsu no Yaiba)**, trazendo uma experiência visual mais envolvente para a consulta de endereços.

O projeto foi desenvolvido como atividade prática do curso de **Análise e Desenvolvimento de Sistemas (ADS)** da **Faculdade Flamingo**, seguindo as orientações apresentadas em aula.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* API de CEP (AwesomeAPI)

---

## 🌐 API Utilizada

Documentação oficial da API:

https://docs.awesomeapi.com.br/api-cep

A API é responsável por retornar informações como:

* CEP
* Logradouro
* Bairro
* Cidade
* Estado

---

## ⚙️ Funcionalidades

* Consulta de CEP em tempo real.
* Exibição automática dos dados do endereço.
* Tratamento de erros para CEPs inválidos.
* Interface temática inspirada em Demon Slayer.
* Consumo de API utilizando JavaScript assíncrono.

---

## 🧠 Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados conceitos importantes de JavaScript, como:

* Consumo de APIs com `fetch()`.
* Programação assíncrona.
* Utilização de `async` e `await`.
* Tratamento de erros com `try` e `catch`.
* Manipulação do objeto `Response`.
* Manipulação do DOM.
* Eventos de formulário.
* Estilização avançada com CSS.
